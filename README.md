# Boston-Housing-Data-Analysis
In the Boston Housing Data Analysis project, I performed exploratory data analysis on the Boston Housing dataset using python. I applied statistical techniques like EDA, Hypothesis testing and regression to understand factors affecting house prices.

Project Title:
Boston Housing Data Analysis

Objective:
Analyze housing prices using statistical methods and visualization.

Tools used:
Python

Pandas

Seaborn

Matplotlib

Analysis Performed:
Descriptive statistics

Boxplots & Histograms

Correlation analysis

Hypothesis testing

Linear regression

Key Insights:
1 - House prices vary significantly by location.
2 - Pollution level and pricing are related to each other.
3 - Value is impacted by student-teacher ratio.
